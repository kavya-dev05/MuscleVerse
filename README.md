## 📌 About

**MuscleVerse** is a comprehensive AI-powered fitness and wellness web platform that helps users take full control of their health journey. From calculating calories to generating fully personalized AI workout and diet plans — MuscleVerse is your all-in-one fitness companion, built with React.js and powered by Generative AI.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Coach** | Generates fully personalized workout & diet plans based on your BMI and fitness goals |
| 🔥 **Calorie Calculator** | Track and calculate daily calorie intake and expenditure accurately |
| 💊 **Supplement Guide** | Curated supplement recommendations tailored to your fitness goals |
| 📏 **Height-Weight Chart** | Visual BMI reference and healthy range tracker |
| 🏆 **Hall of Fame** | Showcase of top fitness achievers to keep you motivated |

## 🧠 How the AI Coach Works

```
User enters age, weight, height & fitness goal
                ↓
         BMI auto-calculated
                ↓
     Generative AI processes profile
                ↓
   Personalized workout plan generated
                ↓
    Custom diet plan tailored to goal
```

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/kavya-dev05/MuscleVerse.git

# Navigate into the project
cd MuscleVerse

# Install dependencies
npm install

# Start the development server
npm run dev
```

> ⚠️ Add your Generative AI API key in a `.env` file before running locally:

```env
VITE_AI_API_KEY=your_api_key_here
```
